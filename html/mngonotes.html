• Developed a Notes App, that can be used to store notes and to do's.
  • Used React Native for frontend and used axios to communicate with API built on PHP .
  • Used MySQL database to store user data in encrypted format.
  • Used AsyncStorage in React Native to store and retrieve data of the logged user when internet connection is not available.
  • Technologies: React.js, React Native, expo.io, JavaScript, MySQL, PHP
